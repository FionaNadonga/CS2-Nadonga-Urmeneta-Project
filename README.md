# PROJECT TITLE: 
    A Story-driven Moral Game

## PROJECT DESCRIPTION: 
    A program that runs a story-driven gambling game where a user must play in a eternal casino and repeat a loop. Wonder why the rich don't give so many to the poor? Wonder why can't all people just be rich? Everything has a value, and it keeps an unfair world a balanced world.

## Features:
    - A story game inspired by a moral lesson: Appreciate that everything has a value.
    - A series of games that enable the player to gain points.
    - A variety of characters and influential events
    - An endless loop, multiple endings.
    - Unique and influential visuals.
## How to run the Program:
    Alt 1:
    1. Log in to any web browser.
    2. Search for the game title.
    Alt 2:
    1. We send the file to you.
    2. You download it, then run.
    ---------------------
    3. If menu appeared, log in your save file(If applicable). Or, just make a new game.
    5. Play the game.
    6. Make sure to save at savepoints, in case the autosave is not in the timeline of your liking.
